# Next.js 版 my-profile

Next.js、TypeScript に置き換えた`my-profile`です。
GithubActions を用いて`main`ブランチに push することで GithubPages にデプロイされます。
パッケージ管理は`yarn`を用いています。

## 導入方法

こちらの Github からクローンし以下コマンドをクローン先のディレクトリで行ってください。

パッケージを導入

```bash
$ yarn
```

dev 環境を起動

```bash
$ yarn dev
```

build をする

```bash
$ yarn build
```
